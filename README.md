👁️ Eye Disease Prediction using SVM

This project is a multiclass eye disease classification system using Support Vector Machine (SVM) and deployed as a Flask web application. Users can upload a retinal image and get the predicted eye disease.

🚀 Features
SVM-based image classification
Upload image & get prediction
Displays uploaded image with result
Simple and user-friendly interface

🛠️ Tech Stack
Python, Flask
Scikit-learn (SVM)
OpenCV, NumPy
HTML, CSS
